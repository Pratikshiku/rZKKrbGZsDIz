### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、事故调查员、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 保险
5. 论坛
6. 公告
7. 个人信息
8. 保险订单
9. 调查申请
10. 事故调查
11. 现场勘查
12. 理赔申请




**事故调查员功能：**
1. 登录
2. 注册
3. 修改密码
4. 个人信息
5. 用户管理
6. 保险管理
7. 调查申请管理
8. 事故调查管理
9. 现场勘查管理
10. 理赔申请管理
11. 公告管理
12. 轮播图管理


**管理员功能：**
1. 登录
2. 个人中心（修改密码、个人信息）
3. 管理员管理
4. 用户管理
5. 事故调查员管理
6. 保险管理
7. 保险订单管理
8. 调查申请管理
9. 事故调查管理
10. 现场勘查管理
11. 理赔申请管理
12. 论坛管理
13. 公告管理
14. 基础数据管理（保险类型管理、公告类型管理、事故调查类型管理、现场勘查类型管理）
15. 轮播图管理
